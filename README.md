# 🎧 2. 서비스 소개

SPEECHFY는 음성 기반으로 작곡을 할 수 있도록 도와주는 인공지능 작곡 도구입니다.
<br>
사용자가 직접 녹음한 허밍이나 음성을 다양한 악기 소리로 변환하고 생성된 사운드를 바탕으로 자동 작곡까지 할 수 있습니다.
<br>
작곡에 필요한 전문 장비나 악기 연주 능력이 없어도 누구나 손쉽게 음악을 만들 수 있는 새로운 창작 환경을 제공하여 유저들이 쉽고 빠르게 곡을 스케치하고 공유할 수 있도록 돕는 데 중점을 두었습니다.

## 🎵 주요 기능

### 🎤 음성을 악기 소리로 변환 (AI 기반 음성 → 악기 변환)

- 사용자가 녹음한 음성(허밍 등)을 인공지능이 학습된 악기 소리로 변환
- 변환 가능한 악기 예시: 바이올린, 트럼펫, 드럼, 색소폰 등
- 사용자는 원하는 악기를 선택하여 녹음 후, 해당 악기 사운드로 자동 변환 가능

### 🎼 AI 기반 자동 작곡

- 변환된 사운드 샘플을 기반으로, 장르(예: 팝, 재즈, 힙합) 및 분위기(예: 따뜻한, 낭만적인, 심플한)를 입력하면 AI가 자동으로 작곡
- 완성된 곡에 어울리는 앨범 커버 이미지도 이미지 생성 AI를 통해 자동 생성

### 🧰 작업실 단위의 프로젝트 관리

- 사용자는 작업실(프로젝트) 별로 사운드를 관리하고 작곡 가능
- 사운드 트랙을 복제하거나 다른 악기로 변환 가능 (예: 바이올린 트랙을 드럼으로 전환)
- 여러 사운드를 조합하여 멀티 트랙 기반의 편곡 가능

### 📤 완성곡 공유 기능

- 완성된 곡을 외부로 공유할 수 있는 기능 제공

### 🕒 메트로놈 기능 (녹음 보조 도구)

- 녹음 시 템포 유지를 위한 메트로놈 제공

## ⭐ 서비스 특장점

### 🎼 작곡 입문자를 위한 낮은 진입장벽

- 전문적인 지식 없이도 허밍만으로 곡을 만들 수 있어 누구나 쉽게 시작 가능

### 🥁 혼자서도 가능한 멀티 악기 편곡

- 한 가지 악기만 다룰 줄 알아도, 다양한 악기 소리로 변환하여 곡 완성 가능

### 🎙 장비 없이 어디서든 작곡 가능

- 별도의 녹음 장비나 연주 환경 없이도, 목소리만으로 작곡 스케치 가능

### 🤖 AI가 돕는 완성도 높은 자동 작곡

- 입력한 장르와 분위기를 기반으로, AI가 자동으로 곡을 완성

### 🎨 커버 이미지까지 자동 생성

- 곡 분위기에 맞는 앨범 커버도 AI가 함께 제작해 완성도를 높임

### 📤 간편한 공유 기능 제공

- 완성된 음악은 누구에게나 쉽게 공유 가능

## 💡 프로젝트의 차별점

### 🎙 음성 기반의 직관적인 작곡 인터페이스

- 대부분의 AI 작곡 서비스는 텍스트 기반 또는 시퀀서 기반의 입력을 요구하지만 SPEECHFY는 '음성(허밍)'만으로 작곡 가능

### 🧠 사운드 생성부터 커버 이미지까지 All-in-One 구성

- 음악 생성, 음원 편집, 앨범 커버 제작, 곡 공유까지 하나의 서비스 안에서 모두 제공
- AI가 창작의 모든 흐름을 지원해 사용자는 하나의 창작 흐름 안에서 몰입감 있게 작업 가능

### 🧩 직접 학습한 AI 모델 & 클라이언트 사이드 처리

- 사용자의 음성을 악기 소리로 변환하는 AI 모델을 직접 학습하여 TensorFlow.js를 통해 클라이언트 단에서 연산되도록 구현
- 서버 부하를 줄이고 빠른 반응성과 높은 확장성을 확보

### 🌐 통신 비용을 줄이는 효율적인 파일 접근 구조

- 음성 파일은 AWS S3에 저장되며, CloudFront를 통해 생성된 접근 주소로
  프론트엔드에서 직접 파일에 접근 가능
- 중간 서버를 거치지 않아 통신 비용을 절감하고 사용자에게 빠른 피드백을 제공 가능

# 7. Conventions

### Branch Convention

- 기능명
  - `feature` : 새로운 기능을 개발
  - `style` : UI/UX 변경
  - `bug` : 에러 해결
  - `refactoring` : 코드 수정
- 각 기능 별 Branch
  > FE-{기능별 브랜치 전략}
  > <br>BE-{기능별 브랜치 전략}
  > <br>ex) FE-feat/login
